Overview

Visit Hibernate App Official Website

Hibernate is an educational storytelling and music app designed for kids, combining fun and learning through interactive audio experiences. I worked closely with two UX designers and one software engineer to implement the explore screen and signup features.

Features

  • Interactive explore screen with category navigation
  • User-friendly signup and onboarding flow
  • Backend integration with Firebase Authentication and Firestore

Technologies Used

Dart, Flutter, Firebase

Challenges

I learned Flutter and Firebase from scratch while integrating wireframes provided by the UX team into responsive and functional app screens. Coordinating with the team remotely also helped me practice communication and version control with Git.

Screenshots

Outcome

The app is expected to launch in late 2025. The current build includes smooth navigation, stylized screens, and a professional user experience built to match high-fidelity UI mockups.
