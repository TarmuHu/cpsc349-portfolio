Overview

A Pygame-based Pong clone featuring multiple scenes, AI paddle control, and full object-oriented design.

Features

  • Scene transitions: Title Scene, Game Scene, and End Scene
  • Human vs AI paddle gameplay
  • Custom soundtrack and sound effects
  • Stylized win/loss quotes
  • PEP-8 compliant Python code

Technologies Used

Python, Pygame

Challenges

Managing clean class-based architecture and implementing responsive AI paddle behavior while maintaining scene separation.
